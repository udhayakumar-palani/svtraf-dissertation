# svtraf-dissertation

## Results and artifacts
All generated artifacts (processed datasets, tables, and publication-ready figures) are consolidated in:

```
svtraf-dissertation/results/
```

- Data: `results/data/` (svtraf_scored.csv, svtraf_dataset.csv, raw reference)
- Tables: `results/tables/` (table_validation_results.csv)
- Figures: `results/figures/` (publication-ready PNGs)

For backward compatibility, the FEW/ folder retains documentation and symlinks to the results location.

See `GETTING_STARTED.md` for reproducible execution instructions and `svtraf-dissertation/FEW/README.md` for dataset notes.
